# Node Graph #

The following is a list of input device shortcuts for manipulating Gaffer's Node Graph and its interface.


## Node Creation ##

```eval_rst
===================================== =============================================
Action			                      Shortcut or Gesture
===================================== =============================================
Show node menu                        :kbd:`Tab` / right-click
Insert `Dot` at connection            Right-click on connection > `Insert Dot`
===================================== =============================================
```


## Node Selection ##

```eval_rst
===================================== =============================================
Action			                      Shortcut or Gesture
===================================== =============================================
Select all                            :kbd:`Ctrl`-:kbd:`A`
Clear selection                       :kbd:`Ctrl`-:kbd:`Shift`-:kbd:`A`
Select node                           Click
Add/remove node from selection        :kbd:`Shift`-click
Select nodes                          Click and drag marquee
Add nodes                             :kbd:`Shift`-click and drag
Select upstream nodes                 :kbd:`Shift`-:kbd:`Alt`-click
Select downstream nodes               :kbd:`Shift`-:kbd:`Ctrl`-click
===================================== =============================================
```


## Node Editing ##

```eval_rst
===================================== =============================================
Action			                      Shortcut or Gesture
===================================== =============================================
Cut node(s)                           :kbd:`Ctrl`-:kbd:`X`
Copy node(s)                          :kbd:`Ctrl`-:kbd:`C`
Paste node(s)                         :kbd:`Ctrl`-:kbd:`V`
Delete node(s)                        :kbd:`Backspace` / :kbd:`Delete`
Enable/disable node(s)                :kbd:`D`
===================================== =============================================
```


## Connections and Layout ##

```eval_rst
===================================== =============================================
Action			                      Shortcut or Gesture
===================================== =============================================
Connect plugs                         Click and drag plug to another plug
Disconnect plug                       Click and drag connection to background
Connect to bookmarked node            Right-click plug > `Connect Bookmark` > select node
Insert node at connection             Drag node onto connection
Auto-arrange selected nodes           :kbd:`Ctrl`-:kbd:`L`
===================================== =============================================
```


## Navigation ##

```eval_rst
===================================== =============================================
Action			                      Shortcut or Gesture
===================================== =============================================
Pan                                   :kbd:`Alt`-click and drag
Zoom                                  :kbd:`Alt`-right-click and drag / mouse wheel
Frame selected nodes                  :kbd:`F`
Enter `Box` node (subgraph)	          :kbd:`↓`
Leave `Box` node (subgraph)           :kbd:`↑`
Search for nodes                      :kbd:`Ctrl`-:kbd:`F`
Bookmark node                         Right-click on node
===================================== =============================================
```
