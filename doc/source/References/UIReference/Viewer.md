# Viewer #

The following is a list of input device shortcuts for manipulating Gaffer's Viewer when previewing 2D images and 3D scenes.


## 2D Image Navigation ##

```eval_rst
===================================== =============================================
Action			                      Shortcut/gesture
===================================== =============================================
Pan                                   :kbd:`Alt`-click and drag
Zoom                                  :kbd:`Alt`-right-click and drag / mouse wheel
Frame image                           :kbd:`F`
Center image at 1:1 scale             :kbd:`Home`
Toggle red channel                    :kbd:`R`
Toggle green channel                  :kbd:`G`
Toggle blue channel                   :kbd:`B`
Toggle alpha channel                  :kbd:`A`
===================================== =============================================
```

## 3D Scene Navigation ##

```eval_rst
===================================== =============================================
Action			                      Shortcut/gesture
===================================== =============================================
Tumble                                :kbd:`Alt`-click and drag
Track                                 :kbd:`Alt`-middle-click and drag
Dolly                                 :kbd:`Alt`-right-click and drag / mouse wheel
Expand selection                      :kbd:`↓`
Fully expand selection                :kbd:`Shift`-:kbd:`↓`
Collapse selection                    :kbd:`↑`
Frame selection                       :kbd:`F`
Fit clipping planes to selection      :kbd:`Ctrl`-:kbd:`K`
Frame and fit clipping to selection   :kbd:`Ctrl`-:kbd:`F`
===================================== =============================================
```
