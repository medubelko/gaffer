# Script Editor #

The following is a list of input device controls and shortcuts for using Gaffer's Script Editor.


## Text Entry ##

```eval_rst
===================================== =============================================
Action                                Shortcut/gesture
===================================== =============================================
Pull node into script                 Middle-click and drag node from Node Graph
                                      onto Script Editor
Pull plug into script                 Middle-click and drag plug from Node Graph or
                                      plug label from Node Editor onto Script Editor
Pull plug value into script           :kbd:`Shift`-click and drag plug label from
                                      Node Editor plug onto Script Editor
Pull color value into script          Click and drag from spot on Viewer onto
                                      Script Editor
Insert path(s) to scene location(s)   Click and drag selection from Viewer or Scene
                                      Hierarchy onto Script Editor
===================================== =============================================
```

## Execution ##

```eval_rst
===================================== =============================================
Action                                Shortcut/gesture
===================================== =============================================
Execute and clear                     :kbd:`Ctrl`-:kbd:`Enter`
Execute selection                     Select code, :kbd:`Ctrl`-:kbd:`Enter`
===================================== =============================================
```
